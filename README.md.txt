                           Mickey AI
The OpenAI-Powered AI Desktop Assistant is a Python-based project that brings the power of OpenAI's GPT-3.5 model to create a human-like conversational experience on your desktop. This assistant, named "Mickey," responds to voice commands, performs various tasks, and engages in natural language conversations.

Prerequisites
Before using the OpenAI-Powered AI Desktop Assistant, ensure you have the following prerequisites installed:
Python 3.x
Required Python packages: speech_recognition, pyttsx3, webbrowser, os, datetime, openai


Getting Started
Clone or download this repository to your local machine.
Obtain your OpenAI API key and add it to the config.py file. Replace the apikey value with your actual API key.

Usage
Run the main.py script using a Python interpreter. This script provides a fully interactive experience with the AI assistant.
The AI assistant listens for voice input and responds with human-like conversation. You can ask it to perform various tasks such as opening websites, providing the current time, playing music, engaging in AI-generated discussions, and more.
To exit the AI assistant, simply say "Mickey Quit."
The conversation history is maintained and expanded throughout the interaction. To start a new conversation, say "reset chat."

Example
Here's a sample interaction with the OpenAI-Powered AI Desktop Assistant:
User: "Hey Mickey, can you open YouTube?"
Mickey: "Sure, opening YouTube for you..."
User: "What's the time right now?"
Mickey: "The current time is [current time]."
User: "Let's discuss artificial intelligence."
Mickey: [Engages in an AI-generated discussion on the topic of artificial intelligence.]
